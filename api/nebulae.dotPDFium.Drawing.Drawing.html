Remarks

The Drawing class includes static methods for creating and manipulating graphical elements such as lines and rectangles on a PDF path object. These methods allow customization of drawing options, including stroke and fill properties, and ensure proper handling of PDF path operations.

Methods

DrawLine(PdfPathObject, DrawingOptions, FsPointF, FsPointF)

Draws a straight line between two points on a PDF path object.

public static void DrawLine(PdfPathObject path, DrawingOptions options, FsPointF start, FsPointF end)

Parameters

path PdfPathObject

The PdfPathObject to which the line will be added. Cannot be null.

options DrawingOptions

The DrawingOptions that specify how the line should be drawn, including whether the path should be closed.

start FsPointF

The starting point of the line, represented as an FsPointF.

end FsPointF

The ending point of the line, represented as an FsPointF.

Remarks

This method modifies the specified path by adding a line segment from the start point to the end point. If the options specify that the path should be closed, the method will attempt to close the path after drawing the line.

Exceptions

ArgumentNullException

Thrown if path is null.

dotPDFiumException

Thrown if the operation to move to the start point, draw the line, or close the path fails.

DrawRect(PdfPathObject, DrawingOptions, FsRectF)

Draws a rectangle on the specified PDF path object.

public static void DrawRect(PdfPathObject path, DrawingOptions options, FsRectF rect)

Parameters

path PdfPathObject

The PdfPathObject on which the rectangle will be drawn. Cannot be null.

options DrawingOptions

The DrawingOptions that specify how the rectangle should be rendered, such as stroke and fill settings.

rect FsRectF

The FsRectF structure defining the rectangle's position and dimensions.

Remarks

This method creates a rectangular path by moving to the top-left corner of the specified rectangle, drawing lines to the other corners, and closing the path. The specified drawing options are then applied to render the rectangle.

Exceptions

ArgumentNullException

Thrown if path is null.

dotPDFiumException

Thrown if an error occurs while drawing the rectangle or closing the path.
