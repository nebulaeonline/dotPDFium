Remarks

The PathBuilder class allows you to create complex paths by chaining method calls such as MoveTo(float, float), LineTo(float, float), and BezierTo(float, float, float, float, float, float). Once the path is fully defined, call Build() to generate a PdfPathObject that can be used in a PDF document. This class enforces proper path construction rules, such as requiring a starting point via MoveTo(float, float) before adding lines or curves. Any violations will result in an exception.
